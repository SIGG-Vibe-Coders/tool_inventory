Connect to GitHub

Paste a GitHub fine-grained personal access token with Contents: Read and write permission on this repository. It is stored only in this browser (never uploaded anywhere) and lets Save to GitHub commit your changes directly.

  1. Open GitHub → fine-grained tokens.
  2. Resource owner: SIGG-Vibe-Coders
  3. Repository access: Only select repositories → tool_inventory
  4. Permissions: Repository permissions → Contents → Read and write
  5. Generate the token, copy it, and paste it below.
